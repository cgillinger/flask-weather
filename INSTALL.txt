📦 FLASK WEATHER DASHBOARD - INSTALLATION & VENV SETUP
=======================================================

SYSTEMKRAV:
-----------
- Python 3.8 eller senare
- pip (Python package manager)
- 50 MB ledigt diskutrymme


FÖRSTA GÅNGEN INSTALLATION:
============================

1. INSTALLERA SYSTEMPAKET (om de saknas):
   -----------------------------------------
   # Ubuntu/Debian:
   sudo apt update
   sudo apt install python3 python3-pip python3-venv

   # Verifiera installation:
   python3 --version    # Ska visa Python 3.8+
   pip3 --version       # Ska visa pip version


2. SKAPA PROJEKTKATALOG OCH VIRTUAL ENVIRONMENT:
   -----------------------------------------------
   cd ~/Dokument/Github/flask_weather
   
   # Skapa venv (första gången):
   python3 -m venv venv
   
   # Aktivera venv:
   source venv/bin/activate
   
   # Du ska nu se (venv) i din prompt:
   # (venv) användare@dator:~/Dokument/Github/flask_weather$


3. INSTALLERA PYTHON-PAKET:
   -------------------------
   # Se till att venv är aktiverat först!
   # (du ska se (venv) i prompten)
   
   pip install --upgrade pip
   pip install -r requirements.txt
   
   # Verifiera installation:
   pip list
   
   # Ska visa:
   # Flask     3.x.x
   # requests  2.x.x
   # + diverse dependencies


4. KONFIGURERA LOCALE (UTF-8 SUPPORT):
   ------------------------------------
   # Kontrollera nuvarande locale:
   locale
   
   # Om LANG är tom eller POSIX, installera Swedish locale:
   sudo locale-gen sv_SE.UTF-8
   sudo update-locale LANG=sv_SE.UTF-8
   
   # Lägg till i ~/.bashrc (valfritt, för permanent fix):
   echo 'export LANG=sv_SE.UTF-8' >> ~/.bashrc
   echo 'export LC_ALL=sv_SE.UTF-8' >> ~/.bashrc
   source ~/.bashrc


5. TESTA INSTALLATION:
   --------------------
   # Se till att venv är aktiverat:
   source venv/bin/activate
   
   # Kör appen:
   python3 app.py
   
   # Öppna i webbläsare:
   http://localhost:8036
   
   # Stoppa appen:
   Ctrl+C

PRODUKTION (SYSTEMD SERVICE):
==============================

Om du vill köra appen som systemd-service för autostart:

1. Skapa service-fil:
   sudo nano /etc/systemd/system/flask-weather.service

2. Innehåll:
   [Unit]
   Description=Flask Weather Dashboard
   After=network.target

   [Service]
   Type=simple
   User=DIN_ANVÄNDARE
   WorkingDirectory=/home/DIN_ANVÄNDARE/Dokument/Github/flask_weather
   Environment="LANG=sv_SE.UTF-8"
   Environment="LC_ALL=sv_SE.UTF-8"
   ExecStart=/home/DIN_ANVÄNDARE/Dokument/Github/flask_weather/venv/bin/python3 app.py
   Restart=always

   [Install]
   WantedBy=multi-user.target

3. Aktivera service:
   sudo systemctl daemon-reload
   sudo systemctl enable flask-weather
   sudo systemctl start flask-weather
   
4. Kontrollera status:
   sudo systemctl status flask-weather
   
5. Stoppa service:
   sudo systemctl stop flask-weather
